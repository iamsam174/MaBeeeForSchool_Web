# MaBeeeForSchool_Web

**MaBeeeForSchool_Web App**は、乾電池型IoTデバイス「MaBeee（マビー） for School」をBluetooth経由で手軽にコントロールするWebアプリです。

PC・タブレット端末・スマートフォンからURL `https://iamsam174.github.io/MaBeeeForSchool_Web/` にアクセスするだけですぐに利用できます。

このアプリはWeb上からBluetoothを操作する技術（Web Bluetooth API）を使用していて、現在 Google Chrome と Microsoft Edge（および一部のAndroidブラウザ）でのみサポートされています。 

Safari や Firefox では動作しませんので、ご注意ください。

⚠️ Appleのセキュリティ方針により、iPad (iPadOS) / iPhone (iOS)のSafari や Chrome ではBluetooth通信機能（Web Bluetooth）がブロックされている関係で、このアプリは動きません。

⚠️ iPad / iPhone で動かしたい場合は「Bluefy」や「WebBLE」といったWeb Bluetooth専用のブラウザアプリでお使いください。

---

## ✨ 主な機能

- **WebGazer.jsによる視線追跡:** インカメラの映像からユーザーの視線をリアルタイムに追跡し、画面上のボタンを見るだけでMaBeeeを操作できます。（※使用するICT機器によっては、インカメラによる視線入力ができない場合があります）
- **多様な動作モード:** 利用者の障害特性や操作の習熟度に合わせて、以下の4つの動作モードを切り替えられます。
  1. **クリックまたはタップでタイマー実行モード:** マウスポインタをボタン内に移動させて左クリックする、または、ボタンをタップすると、タイマー設定した時間（1秒〜180秒）MaBeee for SchoolがONになります。
  2. **注視でタイマー実行（インカメラでの視線入力時）モード:** 設定した秒数（0.1秒〜3.0秒）ボタンを注視すると、タイマー設定した時間（1秒〜180秒）MaBeee for SchoolがONになります。
  3. **マウスオーバー（マウスポインタがボタンに触れた瞬間）でタイマー実行モード:** マウスポインタがボタンに触れた瞬間にタイマーが起動し、一定時間MaBeee for SchoolがONになります。
  4. **マウスポインタがボタン内にある間マビーONモード:** マウスポインタがボタンの領域内にある間だけMaBeee for SchoolがONになり続け、マウスポインタがボタンの外に出るとOFFになります。
- **Web Bluetooth API連携:** 面倒な専用ソフトのインストールは不要です。ブラウザの機能だけで直接MaBeee for SchoolとBluetoothペアリングを行います。
- **UIカスタマイズ:** 対象物の見やすさに合わせて、操作ボタンのサイズや操作音（電子音、ファンファーレなど）を変更可能です。

## ⚠️ 動作環境（必須条件）

ブラウザのセキュリティ制限（Web Bluetooth APIおよびカメラアクセス）のため、以下の環境が必要です。

- **対応ブラウザ:** Google Chrome または Microsoft Edge（※SafariやFirefoxでは動作しません）
- **対応OS:** Windows 10/11, macOS, Android OS, Chrome OS
  - *(※iPad / iPhoneなどのiOS/iPadOS環境では、標準ブラウザSafariがWeb Bluetoothをブロックしているため動作しません。「Bluefy」や「WebBLE」といったWeb Bluetooth専用のブラウザアプリが必要です)*
- **ハードウェア:** 
  - Bluetooth機能（BLE）を搭載したICT機器（WindowsPC、Mac、Android端末、Chromebook）
  - 単４乾電池を入れたMaBeee for School（※青色のMaBeeeとは通信仕様が異なります）および乾電池で作動するオモチャなど
  - Webカメラ（乾電池式オモチャなどの対象物をボタンに映し出したい時に使用します）

## ⚙️ ペアリングと操作手順

1. 画面上の **［MaBeee for Schoolに接続する］** ボタンをクリックします。
2. ブラウザ上部にBluetoothデバイスのリストが表示されるので、「Scratch...」から始まるデバイスを選択して「ペア設定」をクリックします。
3. **［インカメラ］** や **［外付けWebカメラ］** を使う場合、それぞれのボタンを押して、ブラウザからのカメラ使用許可を「許可」にします。
4. 動作モードを設定します。ラジオボタンをクリックして設定するだけでなく、キーボードの［ `1` ］キー［ `2` ］キー［ `3` ］キー［ `4` ］キーでも設定できます。
5. 動作モード①を選択して
6. タイマー秒数を設定します。スライダーをマウスドラッグして動かして設定するだけでなく、キーボードの［ `←` ］キーまたは［ `→` ］キーでも設定できます。
7. 利用者の実態に応じてボタンサイズを切り替えます。ボタンサイズは特大・大・中・小の４つがあり、ドロップダウンメニューから選べます。
8. アプリ上のアクセス内容に応じて音（「ピコーンをという音、または、ファンファーレ）鳴らすことができます。音なしも含めてドロップダウンメニューから選べます。
9. 画面右下の **［🚀 スタート（実行画面へ）］** ボタンを押して実行画面へ移行します。
10. 実行画面に **［🤖 マビーON］** ボタンが表示されますので、それにアクセスするとMaBeee for Schoolを入れた乾電池式オモチャなどが動きます。
11. （視線入力を使用する場合）画面左下の［視線入力: ON / OFF］ボタンで切り替えが可能です。キーボードの［ `X` ］キーでも視線入力のON/OFFが切り替えられます。
